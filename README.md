# NanoTrack 

libtorch版本的NanoTrack跟踪算法，纯C++编写

该项目主要参考[Android $ MacOS version](https://github.com/HonglinChu/NanoTrack),

A Libtorch implementation of the NanoTrack algorithm, written with pure C++. 


## Demo
1. cmake
2. gcc 5.4 +


```
mkdir build && cd build
cmake ..

```

```
make

```

```
./NanoTrack_Libtorch

```
